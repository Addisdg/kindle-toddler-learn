# Generated Sheet Sources

These PNG sheets are AI-generated source art for the toddler flashcard assets.
`generate_assets.py` crops them into individual 400x400 e-ink-friendly PNGs
under `plugins/toddlerlearn.koplugin/assets/`.

The source prompt style was: black-and-white children's flashcard icons, bold
outlines, pure white background, no text labels, no shadows, and one row of
separate subjects.
