# Assets

See `CONTENT_GUIDE.md` in the project root for image specs and how to add
images here.

Subfolders expected by the default `content.lua`:

- `animals/` — cat.png, dog.png, cow.png, bird.png
- `fruit/` — apple.png, banana.png, grapes.png
- `numbers/` and `counting/` — cards from 1.png through 10.png
- `tenframes/` — fixed-grid quantity cards from 1.png through 10.png
- `letters/` — a.png, b.png, c.png
- `shapes/`, `vehicles/`, `body/`, `household/`, `emotions/`
